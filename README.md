# Number to Words

Un semplice script Ruby che converte numeri interi (da 0 a 999) in parole.

## Funzionalità
- Converte numeri da 0 a 999 in parole (es. `123 → "centoventitré"`).

## Tecnologie Utilizzate
- **Ruby**: Linguaggio di programmazione.
